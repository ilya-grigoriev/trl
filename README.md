# trl

`trl` - tool for translating text with Yandex Translate API.

# Installation

For compiling `trl` you need to [pp](https://metacpan.org/pod/pp). You can install it with your package manager.
```
pp -o trl trl.pl
```

Then copy executable file to `bin` folder:
```
cp trl /usr/bin
```
